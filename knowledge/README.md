# База знаний HookahGuide

Структурированная библиотека знаний по кальянной индустрии: оборудование, табак,
забивка, миксология, мастерство и бизнес. Контент хранится в Markdown — по одной
статье на тему, с frontmatter-метаданными. Это **источник истины для текстов**: бэкенд
отдаёт статьи через API (`/api/articles`, `/api/search`), индекс `data/articles.json`
генерируется из frontmatter.

**Сейчас: 37 статей в 13 разделах** (минимум 2 на раздел).

| Папка | Раздел | Что внутри |

## Как наполнять

1. Копируй `_template.md` в нужный раздел, переименуй в `kebab-case-slug.md`.
2. Заполни frontmatter (title, tags, status, level) и тело статьи.
3. Связывай статьи между собой ссылками `[текст](../02-tabak/krepost.md)`.
4. `status: draft` — черновик, `review` — на проверке, `done` — готово.
