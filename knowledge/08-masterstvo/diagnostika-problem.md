# Диагностика проблем: сводная таблица

Быстрый справочник «симптом → причины → что делать». Объединяет частые проблемы
курения. Подробности — в связанных статьях.

## Сводная таблица

| Симптом | Вероятные причины | Решение |
|---------|-------------------|---------|
| **Тяжёлая тяга** | Перетрамбовка; много воды; негерметичность наоборот нет — засор | Рыхлее забить; слить воду до 1–2 см над шахтой; прочистить шахту |
| **Мало дыма** | Мало жара; недобив; мало глицерина; рыхло | Добавить уголь; долить табак; чуть плотнее; больше глицерина |
| **Горечь** | Перегрев; касание фольги; много углей; нагар | Убрать уголь; отступ; почистить калауд; дать «отдохнуть» |
| **Нет вкуса** | Недогрев; грязный прибор; старый табак | Добавить жар; помыть шахту/шланг; свежий табак |
| **Сильно булькает / вода в рот** | Перелив воды; шахта слишком глубоко | Слить воду до погружения 1–2 см |
| **Слабая тяга + «вакуум» не держится** | Негерметичность стыков | Проверить уплотнители, подобрать по размеру |
| **Привкус «химии»** | Саморозжиг; недожжённый уголь | Только полностью разожжённый кокос |
| **Дым горячий** | Мало воды; перегрев; нет диффузора | Долить воду/лёд; снизить жар; поставить диффузор |

## Алгоритм быстрой проверки

1. **Герметичность:** закрой верх шахты ладонью, потяни — «вакуум» должен держаться.
2. **Вода:** шахта погружена на 1–2 см, не больше.
3. **Жар:** недавно положенный уголь полностью разожжён? Сколько углей?
4. **Забивка:** отступ 3–5 мм, без касания, не перетрамбовано?
5. **Чистота:** когда последний раз мыл шахту/шланг? (рекомендуется раз в ~14 дней
   глубокая чистка).

## Профилактика

- Мыть шланг после каждой сессии, прибор — регулярно
  ([уход](../09-uhod/chistka-i-hranenie.md)).
- Не передерживать табак на жаре ([управление жаром](../06-nagrev/upravlenie-zharom.md)).
- Освоить базовую забивку до экспериментов
  ([методы](../04-zabivka/bazovye-metody.md)).
