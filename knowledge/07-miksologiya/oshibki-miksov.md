# Ошибки миксологии: почему микс не получается

Даже из хороших табаков легко собрать «кашу». Вот типичные ошибки и как их избегать —
дополнение к [принципам миксологии](principy-miksologii.md).

## 1. Перегруз вкусами

- **Ошибка:** 4–5+ компонентов в чаше → аромат превращается в неопределённую «кашу».
- **Решение:** 2–3 вкуса максимум. Один ведущий, остальные — поддержка.

## 2. Пропорции 50/50

- **Ошибка:** равные доли часто дают «странный» неопределённый вкус — ни один не лидирует.
- **Решение:** 70/30 или 60/40; для трёх — 60/30/10 или 70/20/10.

## 3. Два сверх-насыщенных вкуса вместе

- **Ошибка:** два интенсивных табака разных фирм → перегруз, борьба вкусов.
- **Решение:** сочетай **интенсивное с более натуральным/нейтральным**.

## 4. Несовместимые семейства

- **Сильный ментол + пряности** → может дать горечь.
- **Цветочное + цитрус** → иногда «парфюмерный», мыльный привкус.
- **Решение:** проверяй спорные пары малыми долями; держись проверенных осей
  (сладкое+кислое, фрукт+фреш).

## 5. Разная влажность табаков

- **Ошибка:** смешал сухой и влажный → неравномерный прогрев (сухой горит, пока влажный
  выпаривает сироп).
- **Решение:** подбирай близкие по влажности; влажный — отжать
  ([состав](../02-tabak/osnova-i-sostav.md)).

## 6. Несбалансированная крепость

- **Ошибка:** ультра-крепкий + лёгкий → вкус и крепость «разваливаются».
- **Решение:** смешивай близкие по крепости линейки; учитывай суммарный никотин
  ([крепость](../02-tabak/krepost.md)).

## Принципы удачного микса (свод)

- **Контраст работает:** сладкое + кислое (малина + лимон).
- **Один лидер** всегда.
- **Фреш/цитрус — приправа**, малыми долями.
- Тестируй и **записывай пропорции** ([шаблон в рецептах](recepty-miksov.md)).
