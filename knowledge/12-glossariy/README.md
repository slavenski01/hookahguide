# Глоссарий кальянных терминов

Словарь терминов и сленга кальянной культуры. Связанные подробные статьи указаны
ссылками.

## Табак

- **Муассель** — самый распространённый кальянный табак: лист, пропитанный сиропом,
  глицерином и ароматизаторами. См. [Основа и состав](../02-tabak/osnova-i-sostav.md).
- **Бестабачка** — смесь без табачного листа (чайная основа, камни, кремы) как
  альтернатива муасселю.
- **Журак (jurak)** — чёрный непромытый табак, очень крепкий и насыщенный.
- **Вирджиния / Берли / Ориентал** — основные сортотипы листа. См.
  [Типы листа](../02-tabak/tipy-lista-i-narezka.md).
- **Отжимка** — удаление лишнего сиропа из влажного табака перед забивкой.

## Оборудование

- **Шахта** — центральная трубка кальяна.
- **Колба** — сосуд с жидкостью.
- **Чаша (чилим)** — ёмкость для табака. См. [Чаши](../05-chashi/vidy-chash.md).
- **Калауд (Kaloud)** — металлическое устройство, заменяющее фольгу: держит угли и
  изолирует их от табака.
- **Колпак / HMD** — устройство управления жаром (heat management device).
- **Диффузор** — перфорированная насадка на конце шахты: дробит дым на мелкие пузыри,
  снижает бульканье и улучшает охлаждение.
- **Уплотнитель** — силиконовое/резиновое кольцо для герметизации стыков.
- **Продувочный клапан** — шариковый клапан для выдувания дыма из колбы.

## Чаши

- **Фанел (funnel)** — чаша с центральным бортиком и одним большим отверстием; держит
  сироп, хороша для крепких/влажных табаков.
- **Турка** — классическая чаша с несколькими отверстиями на дне.
- **Харакири / killer bowl** — чаша с прорезями, под определённые техники.
- **Апгрейд (UpgradeForm)** — популярная чаша из белой глины.

## Забивка

- **Воздушная забивка** — рыхлая укладка без трамбовки.
- **Плотная забивка** — уплотнённая укладка под крепкие табаки.
- **Оверпак (overpack)** — забивка горкой выше краёв с образованием защитной корочки.
  См. [Методы забивки](../04-zabivka/bazovye-metody.md).
- **Касание (халфпак)** — табак вплотную к фольге/калауду (продвинутая техника).
- **Отступ** — зазор между табаком и фольгой/калаудом (обычно 3–5 мм).

## Процесс и сленг

- **Раскур / раскуривание** — прогрев чаши до появления дыма.
- **Прокрутка** — перемешивание/переворачивание табака в процессе для равномерного
  прогрева.
- **Ротация углей** — перестановка углей для управления жаром.
- **Продувка** — выдувание застоявшегося дыма через клапан.
- **Жар** — температурный режим чаши; ключ к вкусу и отсутствию горечи.
- **Микс** — смесь нескольких вкусов. См. [Миксология](../07-miksologiya/principy-miksologii.md).
- **Моно (моновкус)** — один вкус табака без смешивания.
- **Чилл / чильня** — расслабленная атмосфера / кальянная (сленг).

## Угли

- **Кокосовый уголь** — прессованный из кокосовой скорлупы; долго горит, ровный жар.
- **Саморозжиг** — уголь с селитрой, зажигается от зажигалки (даёт привкус, для дома
  нежелателен).
- **Кубик / каб** — единица угля (обычно 22×22 или 25×25 мм).
